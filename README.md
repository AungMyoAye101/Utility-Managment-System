# 🏢 Utility Management System

A full-stack web application for managing tenant utilities, billing workflows, service requests, and property operations. Built with a modern, scalable architecture focusing on clean backend design and maintainable frontend systems.

---

## 🚀 Overview

The **Utility Management System** is designed to streamline property and tenant operations by providing:

* Centralized utility tracking
* Automated billing and invoice management
* Tenant service request handling
* Role-based access control
* Real-time system interaction (planned/extendable)

This project demonstrates strong backend architecture, API design, and production-ready frontend practices.

---

## 🧱 Tech Stack

### 🔹 Frontend (Tenant App / Dashboard)

* React + TypeScript
* Vite
* Tailwind CSS
* React Router
* Axios

### 🔹 Backend (API Server)

* Node.js (ESM)
* Express.js
* Prisma ORM
* PostgreSQL (Neon)
* Zod (validation)
* JWT Authentication

### 🔹 Tooling & DevOps

* tsup (build)
* tsx (dev runtime)
* Prisma migrations & seeding
* Render (deployment)
