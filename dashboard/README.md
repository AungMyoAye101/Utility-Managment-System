# Project Summary
A digital platform for contract and general services management for a housing development. General services include water/electricity issues, monthly payments, and damage & repair complaints.

## Objectives
🧭 Create a centralized management platform
🔍 Increase transparency and trust
⚙️ Facilitate general operations
💬 Simplify communication between homeowners and tenants
📊 Enable accurate decision-making based on reliable data
Included Features
🏢 Manage units/rooms
👥 Manage tenants
📑 Manage contracts/documents
🧾 Automatically generate bills
✉️ Automatically send receipts
🧰 Manage tenant issues and repair requests

Tech Stack
🎨 UI/UX: Figma
⚛️ Frontend: React + HeroUI + ReduxToolkit
🧠 Backend: Express.js
